# PBM Hamming Distance Calculator

Calculate the Hamming distance between two PBM images.

## Build

### Automatic build using `makefile`

```console
make
```

### Manual build using `gcc` compiler

```console
gcc -o main main.c
```

## Usage

```console
./main first-image.pbm second-image.pbm
```
